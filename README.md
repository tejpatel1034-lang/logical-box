# Logic Box 

## Pattern Generator and Number Analyzer

###  Project Description
Logic Box is a Python-based menu-driven application that demonstrates the use of:

- Control Structures
- for and while loops
- Nested loops
- range() function
- break, continue, and pass statements
- Input validation and error handling

This project is designed to strengthen logical thinking and problem-solving skills.

---

##  Features

###  Pattern Generator
- Generates a Right-Angled Triangle pattern
- Uses nested loops
- Takes user input for number of rows
- Validates input

### Number Analyzer
- Accepts a start and end range
- Identifies each number as Odd or Even
- Calculates the sum of all numbers in the range
- Uses range() and loop control statements

###  Menu-Driven Interface
- User-friendly interface
- Continuous execution until Exit is selected
